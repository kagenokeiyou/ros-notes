# 什么是 ROS

机器人操作系统 (Robot Operating System, ROS) 是一个开源的机器人中间件套件。尽管 ROS 不是一个操作系统（OS），而是一套用于机器人软件开发的软件框架，但它提供了为异构计算机集群设计的服务，如硬件抽象、低级设备控制、常用功能的实现、进程间消息传递和包管理。

## 发展历史

### 起源阶段：斯坦福大学（2007 年以前）

ROS 起源于斯坦福大学机器人实验室。博士生 Eric Berger 和 Keenan Wyrobek 在研究个人服务机器人时发现，机器人研究人员在软件、硬件、感知与规划之间存在严重割裂。为降低研究门槛，他们提出构建一个通用、开源的机器人软件基础框架，目标是打造“机器人领域的 Linux”。

### Willow Garage 时期（2007–2013）

2007 年，两位创始人加入 Willow Garage，ROS 正式诞生并于同年首次提交代码。

ROS 与 PR2 机器人 共同发展，逐步形成模块化、分布式的机器人软件生态。

重要里程碑包括：

1. 2010 年发布 ROS 1.0 和首个官方发行版 Box Turtle
2. 向全球高校和研究机构赠送 PR2 机器人，推动 ROS 快速传播
3. ROS 被应用于无人机、自动驾驶汽车和教育机器人
4. 诞生 TurtleBot、ROS Answers 社区与 ROSCon 大会

这一阶段奠定了 ROS 在学术界和工业界的主流地位。

### OSRF / Open Robotics 阶段（2013–至今）

2012 年成立 开源机器人基金会（OSRF），2017 年更名为 Open Robotics，接管 ROS 的核心维护工作。

这一阶段的关键进展：

1. ROS 每年稳定发布新版本，全球社区持续壮大
2. ROS 被用于航天领域（如 NASA 的空间站机器人）
3. 微软、亚马逊等科技公司加入生态，支持 Windows 与云端机器人开发

最重要的突破是 ROS 2：

1. 2014 年提出，2017 年发布首个正式版本
2. 支持实时系统、多平台与工业级应用
3. 标志着 ROS 向下一代机器人操作系统演进
